## MQTT 连接配置

| 项目 | 值 |
|------|-----|
| Host | post-cn-m414bwfgf03.mqtt.aliyuncs.com |
| Port | 1883 |
| Client ID | `GID_AIBuddy@@@{device_id}` |
| 设备 ID | MAC地址（如：E4:B3:23:BF:C0:2C）|

## Topic 结构规范

### 上行 Topic（设备 → 云端）
`aibuddy/{device_id}/{category}`

| category | 功能 |
|----------|------|
| status | 设备状态 |
| msg | 消息互动 |
| ai | AI对话 |
| nfc | NFC功能 |
| study | 学习模式 |
| sensor | 传感器 |
| loc | 定位 |
| update | 资源更新 |

### 下行 Topic（云端 → 设备）
`aibuddy/{device_id}/cmd/{category}`

| category | 功能 |
|----------|------|
| ota | OTA升级 |
| mgmt | 设备管理 |
| msg | 消息下发 |
| info | 信息推送 |
| nfc | NFC指令 |
| config | 配置指令 |

## 错误码定义

| 代码 | 说明 |
|------|------|
| 1001 | 设备离线 |
| 1002 | 设备不存在 |
| 1003 | 目标离线 |
| 1004 | 非好友 |
| 1005 | 请求已存在 |
| 1006 | 好友已满 |
| 1007 | 留言过长 |
| 1008 | 下载失败 |
| 1009 | NFC不存在 |
